# Universal Downloader Telegram Bot 🚀

بوت تلغرام لتحميل الفيديوهات والصور والصوت من أي منصة

## المميزات
- دعم 1000+ منصة (YouTube, TikTok, Instagram, Twitter...)
- تحميل فيديو بجودة عالية أو منخفضة
- تحميل صوت MP3 فقط
- دعم قوائم التشغيل
- واجهة تفاعلية
